Scale, Absolutely!

is extension for RoboFont that allows setting dimensions of selected object in points.

Extreeeme45

adds point at extrema point under 45 degrees on selected curves or the whole glyph.

Add Segment Guideline

is another extension for RoboFont that gives it possibility to add guidelines in a way that Glyphs offers.

sameWidther

same widther is a commandline tool for getting words of same visual width. It can be downloaded from GitHub or via pip. With pip install sameWidther.
